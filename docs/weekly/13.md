# 使用opencode替我完成PPT的制作

使用 OpenCode 制作 PPT，关键是要借助 **“技能（Skills）”** 来扩展它的能力。目前主流的方案有三种，按推荐度排序如下：

### 🥇 方案一：接入 MiniMax Office Skills（最推荐）

这是目前方案最成熟、对中文支持最好、完全免费且效果出色的方案。

这套开源工具链能直接在底层操作文件，保证生成的 PPT 格式专业、不乱码。

**操作步骤：**

1.  **克隆技能库**：在终端中执行以下命令，将官方技能库克隆到本地：
    ```bash
    git clone https://github.com/MiniMax-AI/skills.git ~/.minimax-skills
    ```

2.  **创建配置目录**：确保 OpenCode 的技能配置目录已创建：
    ```bash
    mkdir -p ~/.config/opencode/skills
    ```

3.  **建立软链接**：将下载的技能批量链接到 OpenCode 的目录中：
    ```bash
    ln -s ~/.minimax-skills/skills/* ~/.config/opencode/skills/
    ```

4.  **重启并验证**：**重启 OpenCode**，然后在对话框中输入 `List available skills` 来检查 `pptx-generator` 等技能是否已成功加载。

5.  **开始生成 PPT**：在 OpenCode 对话框中，用自然语言描述你的需求即可。**建议将模型切换为 MiniMax 的 M2.5 或 M2.7 版本，以获得更稳定的效果**。

    **提示词示例**（你可以直接复制并修改内容）：
    > 请调用 pptx-generator 技能，帮我生成一份名为 "Agent_Project_Update.pptx" 的项目汇报幻灯片，保存在当前桌面上。
    > 〖排版与视觉要求〗 风格：科技感、极简专业、商务蓝配色。字体：标题使用微软雅黑加粗，正文使用等线或微软雅黑。布局：注意留白，适当使用项目符号。
    > 〖幻灯片内容大纲〗
    > 第一页（封面）：主标题"多智能体系统与飞书集成项目阶段汇报"，副标题"提升内部自动化协同效率"。
    > 第二页（项目背景与目标）：痛点（缺少统一自动化助理，信息流转效率低），目标（基于开源框架搭建本地多智能体网关，打通企业通讯工具）。
    > 第三页（系统架构设计）：核心大脑（本地部署的大语言模型如 Qwen3-Coder）、调度中心（多智能体网关路由分发）...

### 🥈 方案二：使用 `slidemind` 工具（适合追求品牌设计的用户）

`slidemind` 是一个专为 AI 代理设计的桥接工具，能让 OpenCode 直接操作 PowerPoint。

它的特点是能严格遵循你设定的**企业设计规范**（颜色、字体、布局）来生成 PPT。

*   **安装与初始化**：
    ```bash
    npm install -g slidemind
    slidemind init --for opencode
    ```
*   **前提条件**：需要安装 **Node.js 18+** 和 **Microsoft 365** 或 **PowerPoint 2021/2019/2016**，并在 PowerPoint 中安装 `slidemind` 插件。首次安装还会下载一个约 150-300 MB 的 Chromium 用于渲染。

### 🥉 方案三：安装 `didactic-ppt-skill-package`（备选方案）

这是一个专门用于生成“教学式”PPT 的技能包。

*   **安装**：
    ```bash
    npx --yes @aabadin/didactic-ppt-skill-package
    ```
*   **使用**：重启 OpenCode 后，使用提示词，例如：
    > Create a didactic PowerPoint from this educational source text and save it as a .pptx.

### 💡 通用方法：HTML 转 PPT

如果以上技能都不完全满足需求，还有一个通用方法：**让 OpenCode 先生成一份排版精美的 HTML 代码，然后利用专门的转换工具（如 `html2pptx` 库）将其转为 PPT**。这种方式在样式控制上非常灵活。

### 💎 总结

简单来说，**MiniMax Office Skills** 方案综合体验最好，能帮你快速上手；如果对 PPT 的**品牌视觉规范**有极致要求，可以深入研究 `slidemind`。

建议你先从 **方案一（MiniMax Office Skills）** 开始尝试。配置好后，直接告诉 OpenCode 你的汇报主题和大纲，它就能帮你完成 PPT 的初稿了。
